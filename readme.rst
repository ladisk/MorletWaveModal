MWModal - Morlet-Wave Modal Identification 
------------------------------------------
This is the Python implementation of the Morlet-Wave Modal identification method which is based on the, see [1]_ and [2]_.

This package is created within the MSCA IF project `NOSTRADAMUS`_.
